# 人工签名合成 · 项目功能总结

> 整理时间：2026-07-20。详细施工细则见：`plan.md`（迭代 1）、`plan-迭代2.md`（迭代 2）。

## 1. 项目是什么

macOS 桌面应用（Electron）：把手写签名照片建成「单字/整词图库」，再把表格（xlsx）签名栏里的文字自动替换为白底黑字的签名图，另存导出新表格。解决车间巡查记录表等表单中大量人工签名的录入问题。

代码在 `sign-compose-app/`，双击根目录 `启动签名合成App.command` 即可启动。

---

## 2. 核心使用链路

1. 创建用户、登录（账号密码）
2. 上传签名照片 → 输入签名对应的词 → 逐字拖鼠标画框 → 入库
3. 打开 xlsx 表格 → 填姓名 → 选中单元格点「合成签名」→ 格内文字就地变成签名图
4. 调整签名（缩放/移动/换一张/对齐/删除）→ 确认后导出 `原名_已合成.xlsx`

## 3. 功能清单

### 3.1 多用户

- 注册/登录/切换，每人独立签名库，互不可见
- 密码 scrypt + 盐存储，抗时序比较校验

### 3.2 建库向导

- 上传签名照片（jpg/png 等），输入签名对应的词语
- 逐字画矩形框：拖鼠标、松手确认、可撤销上一框，框序自动对应字序
- 入库统一处理：灰度 → 阈值 180 二值化 → 3×3 膨胀加粗 → 白边自动修剪 → 纯白底纯黑字 PNG
- 单字图存 `chars/`、整词图存 `words/`，`library.json` 索引；一字可多图，默认取第 1 张

### 3.3 字体库管理

- 表格列表视图：缩略图、字/词、拼音、来源词、张数、入库时间
- 搜索：字/词文本、拼音全拼、拼音首字母、来源词，任一命中即显示
- 排序：拼音 / 入库时间 / 张数；过滤：全部 / 仅多图 / 仅单图；单字整词切换
- 删单张（默认图被删自动提升）、设默认、直接上传单字图补库

### 3.4 工作区（所见即所得）

- 打开 xlsx「最大还原」：列宽、行高、合并单元格、对齐、字号、字体族、加粗、字体色、填充色、边框，并显示原表已有嵌入图片；带缩放控件
- 就地编辑：单元格点击改文字；选中若干格 → 合成 → 格内文字就地换成签名图，整表实时呈现
- 合成策略：
  - 整词优先——库里有一模一样的签名直接贴整张；
  - 无整词则单字拼接——统一字高、相邻字重叠 10%、正片叠底融合；
  - 缺字——系统字体混排 + 界面结果列表标红提示具体缺哪个字 + 该格标红
- 签名调整：点选出四角手柄、拖体移动（限单元格区域内）、拖角缩放、右键菜单（换一张字图 / 重生成 / 对齐九宫 / 删除）
- 变形约束：自动放置宽高比锁定（等比充满 96%，不变形）；仅手动拖拽允许自由拉伸
- 删除签名后该格恢复文字可编辑

### 3.5 导出

- 另存 `原名_已合成.xlsx`，原文件字节不动（测试校验哈希）
- 签名图 twoCellAnchor 悬浮锚定，行/列变化随单元格移动；目标格在合并区时按整个合并区域计算
- 原表已有图片、合并单元格、样式全部保留

## 4. 技术架构

| 层 | 选型 |
|---|---|
| 桌面壳 | Electron（主/渲染进程 IPC，17 个 handler） |
| 前端 | 原生 HTML/CSS/JS，无框架 |
| 图像处理 | sharp（二值化/裁剪/缩放/拼接/SVG 文字光栅化） |
| 表格读写 | exceljs（富网格提取、图片锚定插入、另存） |
| 拼音 | pinyin-pro（离线） |
| 存储 | 本地文件目录 + JSON 索引（`userData/users/<id>/`），可直接拷贝备份 |

数据全部在本机，全程无网络请求。

### 代码结构

```
sign-compose-app/
├── src/main/        # 主进程：IPC / 会话 / 文件边界 / 用户 / 图库 / 图像 / xlsx
├── src/preload.js   # contextBridge 安全桥
├── src/renderer/    # 界面：views/users + views/library + views/workspace
└── test/selftest.js # 自测脚本：node test/selftest.js [样例xlsx路径]
```

## 6. 关键设计约束（红线摘要）

- 不做全自动 AI 识别、不做连笔生成、不做复杂表格编辑器
- 不引入 Python 依赖
- 导出永远另存新文件，绝不覆盖原始表格
- 白底黑字全程统一（入库、预览、导出）
- 签名库 = 普通文件 + JSON 索引，可备份可迁移
- 多用户数据完全隔离；数据不出本机
- 开发验收使用临时目录中自动生成的全合成样例，不提交或修改真实业务表格
