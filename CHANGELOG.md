# Changelog

本文件记录面向使用者的版本变化。

## [1.0.0.0] - 2026-07-20

### Added

- 提供离线的签名建库、整词优先合成、单字拼接、缺字提示和 xlsx 另存导出流程。
- 提供账号隔离的本地签名库、签名预览调整和 70 项自包含测试。
- 提供公开仓库 README、ISC 许可证、发布忽略规则和可重复执行的本地测试。

### Changed

- 测试数据改为在系统临时目录动态生成，不再依赖真实业务样例。
- 签名资源改用随机 UUID 文件名，避免在文件系统中暴露姓名或词语。

### Fixed

- 阻止签名索引路径越出当前账号目录，修复跨账号 PNG 覆盖风险。
- 移除固定默认账号和默认密码。
- 转义来自签名库和工作簿的动态 HTML，并禁止应用窗口导航到外部页面。
- 签名图片确认只接受主进程文件选择器最近批准的文件路径。
- 绑定异步文件操作与用户会话代际，防止切换账号时的跨会话竞态。
- 限制密码长度、输入图片大小与像素，以及 xlsx 文件、解压体积和工作表范围。
- 每次从当前内存工作簿快照生成独立导出副本，连续导出不会累加图片，也不会丢失用户的单元格编辑。

### Security

- 发布前敏感信息复检未发现密钥、私人联系方式、真实业务表格或签名图片。
- 生产依赖审计结果为 0 个已知漏洞。
